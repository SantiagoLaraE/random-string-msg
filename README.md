# random-string-msg

Con random-string-msg podrás mostrar mensajes aleatorios.

Fue desarrollado en el Curso de NPM: Gestión de Paquetes y Dependencias en JavaScript de Platzi

## Install

```npm
npm install -g random-string-msg
```

```bash
    random-string-msg
```
